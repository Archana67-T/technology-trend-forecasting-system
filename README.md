
   # Technology Trend Forecasting System

An interactive dashboard that analyzes technology job-demand trends and forecasts future demand using Linear Regression.

## Features
- Technology-wise demand analysis
- Salary trend charts
- Job-posting growth calculation
- Demand forecasting for future months
- Comparison across technologies

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- NumPy

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
